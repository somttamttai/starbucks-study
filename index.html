헷갈리는 음료 차별화 가이드
카페라떼 vs 카푸치노 vs 플랫화이트
카페라떼 (L)
일반 샷 + 우유 + 얇은 폼
부드럽고 크리미
카푸치노 (C)
일반 샷 + 우유 + 두꺼운 폼
X-HOT 스팀!
플랫화이트 (FW)
리스트레또 샷 + 밀크
폼 거의 없음, 진한 맛
이미지 연상 기억법
라떼는 '부드러운 우유 이불' 위에 커피가 살짝 올라간 모습. 카푸치노는 수도사(카푸친 수도회)의 '뾰족한 모자처럼 폼이 높이 솟아오른' 모습. 플랫화이트는 '평평한(flat) 흰 접시'처럼 폼 없이 납작하고 진한 커피.
카라멜 마끼아또 vs 바닐라 라떼
카라멜 마끼아또 (CM)
바닐라시럽 + 우유 + 폼 + 샷(위) + 카라멜드리즐
시럽: 1/2/3/4
바닐라 라떼 (VL)
바닐라시럽 + 샷 + 우유 + 폼
시럽: 2/3/4/5
이미지 연상 기억법
마끼아또는 '얼룩(macchiato)'이란 뜻! 우유 위에 샷을 부어 갈색 '얼룩'이 생기고, 그 위에 카라멜 '그물'이 드리워진 모습을 상상하세요. 바닐라 라떼는 '바닐라 향 가득한 우유 목욕탕'에 커피가 풍덩 들어간 모습!
카페모카 vs 화이트모카
카페모카 (M)
바모카(다크초콜릿) + 샷 + 우유 + 휘핑
진한 초콜릿 색
화이트모카 (WM)
화이트모카(화이트초콜릿) + 샷 + 우유 + 휘핑
밝은 크림색
이미지 연상 기억법
카페모카 = '밤하늘 같은 다크 초콜릿 바다'에 커피가 섞인 모습. 화이트모카 = '눈 덮인 하얀 초콜릿 산' 위에 커피가 흘러내리는 모습. 둘 다 구성 동일, 소스 색만 다르다!
콜드브루 vs 아이스커피 vs 아이스 아메리카노
콜드브루 (CBIC)
찬물 추출 원액 + 물
부드러운 단맛
아이스커피 (IC)
핫 추출 후 냉각
깔끔한 맛
아메리카노 (A)
에스프레소 + 물 + 얼음
샷 맛 강함
이미지 연상 기억법
콜드브루 = 찬 강물에 커피원두를 오래 담가놓은 '천천히 우려낸 냉차'. 아이스커피 = 뜨거운 커피를 '급속 냉동실'에 넣은 느낌. 아메리카노 = 에스프레소에 '폭포수'처럼 물을 부은 모습!
돌체라떼 vs 밀크카라멜라떼
돌체라떼 (SDL)
연유 + 무지방우유 + 폼 + 샷
샷: 1/2/3/4 (사이즈별 다름!)
밀크카라멜라떼 (MCL)
우유 + H&H + 샷 + 솔티드카라멜
두 가지 우유 사용!
이미지 연상 기억법
돌체(dolce=달콤)는 '연유 꿀통'에서 흘러나온 달콤함 + 무지방우유. 밀크카라멜은 '우유 두 겹의 담요(우유+H&H)'를 덮고 소금 카라멜을 뿌린 모습!
에스프레소 마끼아또 vs 에스프레소 콘파나
에스프레소 마끼아또 (EM)
샷 위에 우유 폼
'얼룩' = 하얀 폼 자국
에스프레소 콘파나 (ECP)
샷 위에 휘핑크림
'함께(con) 크림(panna)'
이미지 연상 기억법
마끼아또 = 검은 에스프레소에 '하얀 점(폼) 하나' 찍은 모습. 콘파나 = 에스프레소 위에 '생크림 구름'이 풍성하게 올라간 모습. 점 vs 구름!
