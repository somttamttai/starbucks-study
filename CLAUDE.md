# 스타벅스 공부앱 프로젝트

## 중요 규칙 (반드시 따를 것)
사용자가 수정 요청을 하면 무조건 아래 순서로 진행한다:
1. index.html 파일 수정
2. 수정 완료 후 자동으로 아래 명령어 실행
   git add .
   git commit -m "update: 수정내용요약"
   git push
3. "✅ 배포 완료!" 메시지 출력

## 별도로 배포 요청 안 해도 됨
수정 요청 = 수정 + 배포 자동 실행
